# 🌐 海南大学校园网设计与实现 (Campus Network Design)

## 📖 项目简介
[cite_start]本项目为海南大学《计算机网络》课程的课程设计项目 [cite: 1005, 1006][cite_start]。项目主要使用 Cisco Packet Tracer 软件进行模拟仿真，从底层规划到设备调试，完整设计并实现了一个基于三层交换技术的简单校园网系统 [cite: 1024, 1034]。

## 🛠 技术栈与核心技术
* [cite_start]**仿真平台**: Cisco Packet Tracer [cite: 1024]
* [cite_start]**主要网络设备**: 3560-24 系列三层交换机、2950-24 系列二层交换机、2621XM / 2620XM 系列路由器、各类型服务器与 PC [cite: 1029, 1134, 1144]
* **关键网络技术**:
  * [cite_start]**三层交换与 VLAN**: 运用三层交换机实现二层交换与网络层转发的结合，解决了传统路由器导致的低速瓶颈问题 [cite: 1064]。
  * [cite_start]**路由协议**: 综合使用了静态路由以及 RIP 和 OSPF 动态路由协议来实现跨网段通信 [cite: 1029, 1277, 1291]。
  * [cite_start]**网络地址转换 (NAT)**: 实施动态 NAT 和端口重载 (Overload)，满足校内私有地址访问外部网络的需求 [cite: 1024, 1281, 1289]。
  * [cite_start]**生成树协议 (Spanning Tree)**: 利用生成树算法避免链路环路，抑制网络风暴 [cite: 1325]。

## 🏗 网络架构与拓扑规划
[cite_start]本项目对校园网中的主要建筑物（如行政楼、教学楼、图书馆、宿舍等）进行了网络拓扑设计 [cite: 1091]。
* **VLAN 子网划分**: 为了方便管理并增强安全性，将校园网按照部门划分为了 8 个相互独立的 VLAN：
  * [cite_start]办公室 (VLAN 1)、机房 (VLAN 2)、自习室 (VLAN 3)、楼管室 (VLAN 4)、寝室 (VLAN 5)、多媒体教室 (VLAN 6)、服务器区 (VLAN 7)、会议室 (VLAN 8) [cite: 1114]。
* [cite_start]**IP 地址分配**: 校内采用 C 类私有 IP 地址（从 `192.168.0.0/16` 段中选取），各 VLAN 被分配了不同的独立网段，并为每个网段指定了网关 [cite: 1036, 1122]。

## 📊 测试与结论
[cite_start]项目对网络原型的可用性进行了全面的测试和诊断 [cite: 1045, 1645]。
* [cite_start]**连通性测试**: 通过 `ping` 命令和 Web 浏览器访问，验证了交换机与终端、内网与内网、内网与外网之间的通信情况 [cite: 1699, 1916, 1973]。
* [cite_start]**运行结果**: 测试表明，校园网内部不同 VLAN 的主机和服务器之间可以自由通信，且内外网之间的数据传输正常，证实了该网络设计方案的成功与可用性 [cite: 1994]。
